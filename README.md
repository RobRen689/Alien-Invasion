# Alien-Invasion

A simple space shooting game made with Python using the pygame module.

The player controls a space ship located at the bottom of the screen. The player can move the ship right and left by using the corresponding arrow keys and shoot bullets using the spacebar. When the game start, a fleet of alien will start to descend the screen. The player will be required to shoot the aliens. If all aliens are destroyed, a new fleet will appear but will drop down at a faster rate to increase difficulty. If any alien hits the player or the alien hits the bottom of the screen, the player will lose a life. If all three lifes are lost, the game is over.
